- **Tasks board** — `tasks-board.png` — Kanban-style task columns on the Tasks page.

## Adding new screenshots

1. Put images in `docs/images/`.
2. Reference in markdown as: `![Description](images/filename.png)`.
3. Update this file and `docs/images/README.md` with the new file and where it’s used.
